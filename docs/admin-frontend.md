# 管理面板文档

## 概述

管理面板为酒店管理员提供后台管理功能，所有管理页面受 `AdminRoute` 守卫保护（需登录且 `role === 'admin'`）。

## 入口

- 路径：`/admin`
- 登录页：`/admin/login`（深色背景，区别于用户登录）
- 导航：登录后 Navbar 显示"管理面板"链接（仅管理员可见）

## 页面列表

### AdminLogin — 管理员登录

| 项目 | 说明 |
|------|------|
| 路径 | `/admin/login` |
| 守卫 | 无（独立登录页） |
| 功能 | 用户名 + 密码登录，验证 `role === 'admin'` |
| 登录后 | 跳转 `/admin` |
| 非管理员 | 提示"该账号不是管理员" |

### Dashboard — 仪表盘

| 项目 | 说明 |
|------|------|
| 路径 | `/admin` |
| 守卫 | AdminRoute |
| API | `GET /stats` |

**统计卡片（6 项）**

| 指标 | 图标 | 颜色 |
|------|------|------|
| 总用户 | 👥 | 蓝色 |
| 总房间 | 🏨 | 绿色 |
| 可用房间 | ✅ | 绿色 |
| 总预订 | 📋 | 橙色 |
| 待确认 | ⏳ | 黄色 |
| 总收入 | 💰 | 金色 |

**预订统计条形图**

- 已确认、待确认、已入住、已退房、已取消
- 每种状态独立颜色

**房型收入分布**

- 按房型分组显示收入和订单数

**最近预订表格**

- 展示最近 5 条预订：用户、房间、入住/退房日期、金额、状态

**快捷链接**

- 管理房间、管理预订、管理用户

**数据导出**

- 导出预订 CSV、导出用户 CSV、导出房间 CSV
- URL 格式：`/api/export/:type?token=<JWT>`

### AdminRooms — 房间管理

| 项目 | 说明 |
|------|------|
| 路径 | `/admin/rooms` |
| 守卫 | AdminRoute |
| API | `GET /rooms`, `POST /rooms`, `PUT /rooms/:id`, `DELETE /rooms/:id`, `POST /upload` |

**搜索**

- 按房间名称搜索，输入即时查询

**添加/编辑表单**

| 字段 | 类型 | 说明 |
|------|------|------|
| 房间名称 | text | 必填 |
| 房型 | select | 标准/豪华/套房/总统套房/家庭/商务 |
| 价格/晚 | number | 必填 |
| 容纳人数 | number | 必填 |
| 描述 | textarea | — |
| 设施 | text | 逗号分隔 |
| 房间图片 | file | 通过 `POST /upload` 上传，仅 jpg/png/gif/webp，5MB |
| 状态 | select | 可预订 / 不可订 |

编辑时表单自动填充现有数据，取消编辑恢复空表单。

**房间列表表格**

| 列 | 说明 |
|----|------|
| ID | 自增 ID |
| 图片 | 缩略图或 `-` |
| 名称 | — |
| 类型 | 中文标签 |
| 价格 | ¥ + 价格 |
| 容量 | N人 |
| 评分 | 星级 + 分数或 `-` |
| 状态 | ✅ / ❌ |
| 操作 | 编辑 / 删除 |

删除时级联删除该房间的评价和预订。

**分页**：每页 10 条

### AdminBookings — 预订管理

| 项目 | 说明 |
|------|------|
| 路径 | `/admin/bookings` |
| 守卫 | AdminRoute |
| API | `GET /bookings`, `PUT /bookings/:id`, `DELETE /bookings/:id` |

**筛选栏**

| 按钮 | 说明 |
|------|------|
| 全部 | 无状态筛选 |
| 待确认 | pending |
| 已确认 | confirmed |
| 已入住 | checked_in |
| 已退房 | checked_out |
| 已取消 | cancelled |

**搜索**：按用户名或房间名搜索

**导出 CSV**：按当前筛选条件导出，`window.open` 新窗口下载

**预订列表表格**

| 列 | 说明 |
|----|------|
| ID | — |
| 用户 | 用户名 |
| 房间 | 房间名 |
| 入住 | 格式化日期 |
| 退房 | 格式化日期 |
| 人数 | — |
| 金额 | ¥ + 价格 |
| 状态 | 彩色 Badge |
| 操作 | 状态按钮 + 删除 |

**状态流转按钮**

| 当前状态 | 可用操作 | 按钮颜色 |
|----------|----------|----------|
| pending | 确认 | 绿色 |
| confirmed | 入住 | 蓝色 |
| checked_in | 退房 | 灰色 |
| 非 checked_out/cancelled | 取消 | 黄色 |
| 任意 | 删除 | 红色 |

状态变更时自动推送站内通知给预订用户。

**分页**：每页 15 条

### AdminUsers — 用户管理

| 项目 | 说明 |
|------|------|
| 路径 | `/admin/users` |
| 守卫 | AdminRoute |
| API | `GET /users`, `PUT /users/:id`, `DELETE /users/:id` |

**搜索**：按用户名或邮箱搜索

**导出 CSV**：导出所有用户数据

**用户列表表格**

| 列 | 说明 |
|----|------|
| ID | — |
| 用户名 | — |
| 邮箱 | — |
| 角色 | 管理员（金色 Badge）/ 用户（灰色 Badge） |
| 注册时间 | — |
| 操作 | 编辑 / 删除 |

**行内编辑**：点击"编辑"后该行变为输入框模式，可修改用户名、邮箱、角色，保存或取消。

| 编辑字段 | 类型 | 说明 |
|----------|------|------|
| 用户名 | text | — |
| 邮箱 | text | — |
| 角色 | select | 用户 / 管理员 |

**删除规则**

- 无法删除管理员账户（后端 403 拒绝）
- 删除用户时级联删除其通知、评价、预订

**分页**：每页 15 条

## 通用交互模式

| 交互 | 方式 |
|------|------|
| 操作反馈 | Toast 通知（`useToast`） |
| 加载状态 | `<Loading />` 组件 |
| 删除确认 | `window.confirm` 弹窗 |
| 分页 | `<Pagination />` 组件 |
| 表格滚动 | `.table-wrap` CSS 横向滚动 |
| 返回面板 | 每个管理页顶部"返回面板"链接 |
